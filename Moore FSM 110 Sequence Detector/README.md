# Project 4 - Moore FSM 110 Sequence Detector

## Overview

입력 데이터에서 "110" 패턴을 검출하는 Moore FSM을 구현한 프로젝트입니다.

## Objective

* Sequence Detector 설계
* Moore Machine 상태 전이 이해

## Learning Points

* 상태 전이도 작성
* 출력과 상태의 관계 분석
* 패턴 검출 알고리즘 구현

## Skills Acquired

* FSM Design
* Pattern Detection
* RTL Verification
