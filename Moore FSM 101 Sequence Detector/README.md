# Project 3 - Moore FSM 101 Sequence Detector

## Overview

Moore FSM(Finite State Machine)을 이용하여 입력 비트 스트림에서 "101" 패턴을 검출하는 회로를 설계한 프로젝트입니다.

## Objective

* Moore Machine 구조 이해
* 상태 전이(State Transition) 설계
* Sequence Detector 구현

## Learning Points

* FSM 상태도 작성
* State Encoding
* Moore 출력 구조 이해

## Skills Acquired

* FSM Design
* Sequential Logic Design
* Verilog RTL Coding
* Testbench Verification
