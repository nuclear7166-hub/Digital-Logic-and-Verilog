# Project 24 - Binary to BCD Converter

## Overview

Binary 데이터를 BCD(Binary Coded Decimal) 형식으로 변환하는 회로를 설계한 프로젝트입니다.

FND(7-Segment Display) 출력의 전처리 과정으로 사용됩니다.

---

## Objective

- Binary ↔ Decimal 변환 이해
- Display Interface 학습
- 데이터 포맷 변환 구현

---

## Example

### Binary

```text
10110110
```

### Decimal

```text
182
```

### BCD

```text
0001 1000 0010
```

---

## Key Concepts

### Double Dabble Algorithm

Shift와 Add-3 연산을 이용한 BCD 변환

### Display Processing

FND Controller와 연동 가능
