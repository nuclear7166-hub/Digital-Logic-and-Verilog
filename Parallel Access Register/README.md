# Project 18 - Parallel Access Register

## Overview

병렬 데이터 접근이 가능한 Register를 설계하여 다중 비트 데이터를 효율적으로 저장하고 읽어오는 구조를 학습한 프로젝트입니다.

---

## Objective

- Register File 개념 이해
- 병렬 데이터 처리 학습
- 다중 비트 저장 구조 설계

---

## Key Concepts

### Parallel Access

```text
Input[7:0]
    │
    ▼
 Register
    │
    ▼
Output[7:0]
```

모든 비트를 동시에 접근할 수 있습니다.
