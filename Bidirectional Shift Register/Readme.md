# Project 11 - Bidirectional Shift Register

## Overview

좌측 및 우측 방향으로 데이터를 이동시킬 수 있는 Bidirectional Shift Register를 설계한 프로젝트입니다.

데이터 이동 방향을 제어 신호를 통해 선택할 수 있습니다.

---

## Objective

- 양방향 데이터 이동 구현
- Shift Register 확장 구조 학습
- 제어 신호 기반 동작 설계

---

## Key Concepts

### Shift Left

```text
1010 → 0100
```

### Shift Right

```text
1010 → 0101
```

### Direction Control

```text
DIR = 0 → Left Shift
DIR = 1 → Right Shift
```
