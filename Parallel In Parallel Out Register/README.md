# Project 13 - Parallel In Parallel Out Register

## Overview

병렬 데이터를 동시에 입력받고 동시에 출력하는 PIPO Register를 설계한 프로젝트입니다.

고속 데이터 저장 및 전달에 활용됩니다.

---

## Objective

- Parallel Data 처리 이해
- 다중 비트 저장 방식 학습
- Register 확장 설계

---

## Key Concepts

### PIPO Structure

```text
Input[3:0]
     │
     ▼
 PIPO Register
     │
     ▼
Output[3:0]
```

### Parallel Data Transfer

모든 비트를 동시에 저장 및 출력합니다.
