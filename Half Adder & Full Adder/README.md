# Project 2 - Half Adder & Full Adder

## Overview

Half Adder와 Full Adder를 설계하고 이를 확장하여 4-bit Ripple Carry Adder를 구현한 프로젝트입니다.

## Objective

* 이진 덧셈 원리 이해
* 조합논리회로 설계
* Carry 전파 과정 이해

## Files

| File  | Description      |
| ----- | ---------------- |
| ha.v  | Half Adder       |
| fa.v  | Full Adder       |
| fa4.v | 4-bit Full Adder |

## Learning Points

### Half Adder

두 비트의 합(Sum)과 자리올림(Carry) 생성

### Full Adder

Carry 입력을 포함한 덧셈 회로

### Ripple Carry Adder

Full Adder 여러 개를 연결하여 다비트 덧셈 구현

## Skills Acquired

* XOR 게이트 활용
* Carry Chain 이해
* 계층적 설계(Hierarchical Design)
