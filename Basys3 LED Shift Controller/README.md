# Project 22 - Basys3 LED Shift Controller

## Overview

Basys3 FPGA 보드의 LED를 순차적으로 점등하는 Shift Controller를 구현한 프로젝트입니다.

---

## Objective

- FPGA I/O 제어 학습
- LED 출력 제어
- Shift Logic 구현

---

## Key Concepts

### LED Shift Pattern

```text
00000001
00000010
00000100
00001000
...
```

### Visual Verification

LED 점등 패턴을 통해 회로 동작 확인
