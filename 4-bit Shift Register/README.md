# Project 10 - 4-bit Shift Register

## Overview

4-bit Shift Register를 설계하여 데이터를 순차적으로 이동시키는 방법을 학습한 프로젝트입니다.

Shift Register는 직렬 데이터 처리 및 통신 인터페이스(UART, SPI 등)의 기본 구성 요소로 사용됩니다.

---

## Objective

- Shift Register 동작 원리 이해
- 직렬 데이터 이동 방식 학습
- Sequential Logic 설계 경험 습득

---

## Key Concepts

### Shift Operation

클록 신호가 입력될 때마다 데이터가 한 비트씩 이동합니다.

```text
Before
1010

After Shift Right
0101
```

### Serial Data Transfer

데이터를 한 비트씩 순차적으로 전송할 수 있습니다.
