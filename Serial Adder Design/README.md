# Project 19 - Serial Adder Design

## Overview

Shift Register와 Full Adder를 이용하여 Serial Adder를 구현한 프로젝트입니다.

하나의 Full Adder를 재사용하여 여러 비트 연산을 수행하는 구조를 학습하였습니다.

---

## Objective

- Serial Arithmetic 이해
- Carry 저장 방식 학습
- 하드웨어 자원 최적화 설계

---

## Key Concepts

### Serial Adder Architecture

```text
A Register ----\
                \
                 Full Adder ---> Sum
                /
B Register ----/

Carry Register
```

### Carry Storage

이전 연산의 Carry를 저장하여 다음 연산에 활용합니다.
