# Digital-Logic-and-Verilog

# Verilog Study

Verilog HDL을 이용한 디지털 회로 설계 및 FPGA 실습 프로젝트를 정리한 저장소입니다.

기초 조합논리회로부터 FSM(Finite State Machine), 레지스터, 카운터, 메모리(SRAM), FND 제어, 모터 제어까지 단계적으로 학습하였습니다.
