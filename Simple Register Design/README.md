# Project 12 - Simple Register Design

## Overview

기본 Register를 설계하여 클록 기반 데이터 저장 원리를 학습한 프로젝트입니다.

디지털 시스템에서 레지스터는 데이터를 임시 저장하는 가장 기본적인 순차 논리회로입니다.

---

## Objective

- Register 동작 이해
- Clock Edge Trigger 학습
- 데이터 저장 구조 구현

---

## Key Concepts

### Register

```text
Input Data
    │
    ▼
 Register
    │
    ▼
Output Data
```

### Clock Synchronization

클록 상승 에지에서만 데이터가 저장됩니다.
