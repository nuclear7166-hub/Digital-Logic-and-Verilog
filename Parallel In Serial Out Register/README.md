# Project 17 - Parallel In Serial Out Register

## Overview

PISO(Parallel In Serial Out) Register를 설계하여 병렬 데이터를 직렬 데이터로 변환하는 방법을 학습한 프로젝트입니다.

---

## Objective

- 병렬 데이터를 직렬 데이터로 변환
- 데이터 직렬화(Serialization) 이해
- 통신 인터페이스 기초 학습
